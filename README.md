# EnMem-Rekognition

AWS Rekognition Chalice

## Supported File Extensions

* Image: JPEG, JPG, PNG
* Video: MOV, MPEG-4, FLV (Based on H264)

## Supported Function

* Labels(identify thousands of objects)
* Face detection and analysis(emotion, gender, smiling etc)
